# TSP Solution using Python 3

To run this file in terminal:

  -   Open command line and change direction to this project (TSP folder).
  
  > **cd your/path/TSP/**
  
  -   Run with python 3;
  
  > **python  src/nearest_neighbour.py**
  
# Environment:
  
  - OS: Ubuntu 19.04
  
  - Python: 3.7.3
